Sportify Database Migrations

Cảnh báo: Thư mục này chứa các file migration database. Không truy cập hoặc sửa đổi trực tiếp!

Đây là thư mục chứa các script SQL để cập nhật cấu trúc database. Các file này được thực thi tự động bởi Flyway khi ứng dụng khởi động.

Các file migration phải tuân theo quy tắc đặt tên: V{version}__{description}.sql
